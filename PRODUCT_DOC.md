# transread-pdf 产品文档

## 1. 产品简介

transread-pdf 是一款 Chrome 扩展，用于将浏览器中的 PDF 内容转换为可翻译、可搜索、可批注的网页阅读器界面，核心目标是让用户像读网页一样使用 Chrome 自带翻译能力阅读 PDF，同时保留 PDF 阅读器的常用能力（目录、跳页、缩放、批注、打印、下载）。

## 2. 目标用户与使用场景

- 需要阅读外文 PDF（论文、书籍、白皮书、报告）并依赖浏览器翻译能力的用户
- 需要在 PDF 中快速检索关键词并定位上下文的用户
- 需要做简单标注（高亮、批注、涂鸦、插图）的用户

典型场景：
- 读论文：段落级双语对照 + 搜索关键词 + 高亮重点
- 读书：沉浸式阅读主题 + 目录导航 + 跳页
- 做笔记：文本备注 + 画笔标注 + 插图截图

## 3. 核心价值

- **翻译友好**：将 PDF 文本组织为网页段落结构，使 Chrome 翻译对阅读体验更友好
- **可读性强**：段落级中英对照（英文保持原文，中文由网页翻译生成）
- **高效率定位**：全文搜索 + 上一处/下一处跳转 + 高亮匹配
- **阅读器能力补齐**：目录导航、跳页、缩放、主题切换
- **轻量批注**：高亮、文本备注、画笔涂鸦、插入图片

## 4. 功能清单（当前版本）

### 4.1 打开与转换

- 支持在 PDF 页面点击插件图标打开阅读器页面
- 支持手动上传本地 PDF 文件进行转换（避免跨域限制）
- 支持解析并展示 PDF 目录（Outline）

### 4.2 阅读与导航

- 左侧目录侧边栏显示/隐藏
- 页码输入跳转
- 缩放（50%～200%）
- 主题切换：深色 / 浅色 / 护眼（Sepia）

### 4.3 搜索

- 工具栏搜索按钮打开搜索面板
- 输入关键字后高亮全文匹配
- 支持上一处/下一处跳转
- 支持“区分大小写”

### 4.4 批注与编辑

- 高亮：选中文本后点击高亮按钮
- 文本备注：选择文本工具后点击页面放置可编辑备注
- 画笔涂鸦：选择画笔工具后在页面上绘制
- 插入图片：选择图片工具后从本地选择图片并插入，可拖动、双击删除

### 4.5 打印与下载

- 打印/导出：调用浏览器打印能力（打印时隐藏工具栏与侧边栏）
- 下载：下载原始 PDF 文件（若来源允许）

## 5. 权限与安全说明

- 扩展需要访问当前标签页 URL，用于将 PDF 链接传入阅读器页面
- 若用户需要读取本地文件（file://），需要在扩展详情页开启“允许访问文件网址”
- 不会上传 PDF 内容到服务器；解析与渲染在浏览器本地完成

## 6. 交互说明（关键路径）

### 6.1 打开在线 PDF

1. 打开一个 PDF 页面
2. 点击扩展图标
3. 新标签页打开阅读器并自动加载内容
4. 右键选择“翻译成中文”，得到段落级双语对照

### 6.2 打开本地 PDF / 受限 PDF

1. 点击扩展图标打开阅读器页面
2. 点击“选择 PDF 文件”上传
3. 渲染成功后可进行翻译、搜索与批注

## 7. 已知限制

- 某些 PDF 的文本流非常碎（尤其是扫描版或复杂排版），段落合并效果可能不完美
- 图片提取与“精确插入段落之间”的一致性受 PDF 内部绘制指令影响，部分 PDF 可能只能保证“图片可见”，但插入位置可能不完全精准
- 下载功能受原始 PDF 地址的跨域与权限策略影响

## 8. 后续迭代方向（建议）

- 搜索支持：正则、全词匹配、仅搜索译文/仅搜索原文
- 批注导出：将批注保存为 JSON，并支持再次打开恢复
- 高亮管理：批注列表、跳转到批注位置
- 更稳定的图片定位：支持更多绘图指令类型、旋转与裁剪坐标映射
