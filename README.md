# PDF 转网页 (翻译助手)

这个谷歌浏览器插件可以帮助你将 PDF 文件动态转换为普通的 HTML 网页（即 index），从而能够使用 Chrome 自带的网页翻译功能对整个 PDF 内容进行翻译。

## 核心功能

- **一键转换**：在任意 PDF 页面点击插件图标，即可自动解析内容。
- **排版优化**：按页提取文本，自动分段排版，方便阅读。
- **支持谷歌翻译**：转换后得到的是标准 HTML 文本，右键点击“翻译成中文”即可无缝使用 Chrome 自带翻译功能。

## 安装说明（开发者模式）

1. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
2. 在右上角开启 **“开发者模式”** 开关。
3. 点击左上角的 **“加载已解压的扩展程序”** 按钮。
4. 在弹出的窗口中，选择当前这个 `pdf-index` 文件夹。

## 使用方法

1. 在浏览器中打开任意一个 PDF 文件（可以是在线的网络链接，也可以是本地的 PDF 文件）。
2. 点击浏览器右上角插件栏中的 **“PDF 转网页”** 插件图标（建议将其固定在工具栏）。
3. 插件会打开一个新标签页，将 PDF 的文本内容提取并渲染为普通网页。
4. 在新标签页中，右键点击空白处，选择 **“翻译成中文”** 即可完成翻译！

> **⚠️ 重要注意：如果需要翻译本地的 PDF 文件**
> 如果你打开的是本地文件（地址栏以 `file:///` 开头），默认情况下 Chrome 会限制插件读取它。
> **解决方法**：请在插件管理页面（`chrome://extensions/`）找到该插件，点击“详细信息”，向下滚动并开启 **“允许访问文件网址”** 选项。

## 技术栈
- Manifest V3
- PDF.js (解析核心)
